# 0x03. Shell, init files, variables and expansions

This directory contains solutions to tasks related to shell initialization files, variables, and expansions.

## Scripts Description

*   **0-alias**: A script that creates an alias. It sets `ls` to `rm *`.
*   **1-hello_you**: A script that prints "hello user" where "user" is the current Linux user.
*   **2-path**: A script that adds `/action` to the `PATH` environment variable. `/action` becomes the last directory the shell looks into.
*   **3-paths**: A script that counts the number of directories in the `PATH` environment variable.
*   **4-global_variables**: A script that lists all environment variables.
*   **5-local_variables**: A script that lists all local variables, environment variables, and shell functions.
*   **6-create_local_variable**: A script that creates a new local variable named `BEST` with the value `School`.
*   **7-create_global_variable**: A script that creates a new global variable named `BEST` with the value `School`.
*   **8-true_knowledge**: A script that prints the result of 128 plus the value of the `TRUEKNOWLEDGE` environment variable.
*   **9-divide_and_rule**: A script that prints the result of `POWER` divided by `DIVIDE`.
*   **10-love_exponent_breath**: A script that displays the result of `BREATH` to the power of `LOVE`.
*   **11-binary_to_decimal**: A script that converts a binary number (stored in the `BINARY` environment variable) to base 10.
*   **12-combinations**: A script that prints all possible combinations of two lowercase letters, excluding "oo". The output is alpha-ordered.
*   **13-print_float**: A script that prints a number (from the `NUM` environment variable) formatted to two decimal places.
*   **100-decimal_to_hexadecimal**: A script that converts a decimal number (stored in the `DECIMAL` environment variable) to hexadecimal.
*   **101-rot13**: A script that encodes and decodes text using the rot13 encryption.
*   **102-odd**: A script that prints every other line from the input, starting with the first line.
*   **103-water_and_stir**: A script that adds two numbers stored in environment variables `WATER` and `STIR` and prints the result. `WATER` is in base `water`, `STIR` is in base `stir.`, and the result is in base `bestchol`.
